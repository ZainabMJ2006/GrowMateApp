# GrowMate App 🌱
GrowMate is a Java desktop application designed to help users manage and take care of their plants through an organized and user-friendly system.

## Features
- User login and sign-up system
- Add and manage plants
- Store plant information
- Simple graphical user interface (GUI)

## Technologies Used
- Java
- Java Swing (GUI)
- Object-Oriented Programming (OOP)

## Project Structure
- `src/growmate/` → main application files  
- `dao/` → data access layer  
- `model/` → data models  
- `db/` → database connection  

## How to Run
1. Open the project in NetBeans
2. Run `GrowMateApp.java`
3. Use the interface to manage plants
